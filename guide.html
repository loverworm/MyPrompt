📘 功能说明

✅ 支持网站：Deepseek、豆包、Kimi

插件主界面（管理器）
浮动快捷面板（按 Alt+1 唤出）

🖱️ 界面与按钮说明

扩展主窗口分为左侧导航栏和右侧内容区。以下是各按钮的功能详解：

📌 左侧导航栏

支持分组折叠（核心/管理/系统/帮助组），状态自动记忆。

按钮名称功能说明
🗔 全屏模式位于左侧导航栏顶部，点击可在独立标签页中打开完整管理界面，获得更大操作空间。
📋 提示词列表默认页面，展示所有已启用（最新版本）的正式词条。支持多种排序方式（见下文）。
⭐ 我的收藏展示您收藏的提示词，支持取消收藏、填入、复制、编辑、删除。注意：此处的"删除"按钮会彻底删除词条（从所有列表中移除），而非仅取消收藏。收藏词条在主列表中会以浅蓝色背景+左侧蓝色边框高亮显示，并在卡片右上角显示"已收藏"文字标签（若同时置顶则优先显示"已置顶"）。
📝 草稿箱存放未完成的草稿，可继续编辑或转为正式词条。转为正式时，会自动禁用所有同名的旧正式词条（保留在版本历史），并创建新的正式版本。
📁 分类管理添加、删除自定义分类，并可为每个分类选择背景色（15种预设颜色）。
🏷️ 标签管理添加、删除自定义标签。
📜 版本历史独立页面，以时间轴展示每个提示词的所有版本，支持恢复任意旧版本。恢复时，该标题下的所有旧版本会被自动取消收藏和置顶，恢复的版本本身也不带收藏/置顶状态。删除当前生效版本时，若该词条系列下还存在其他历史版本，系统会自动将最新历史版本恢复为生效状态，防止误删导致数据完全丢失。页面顶部新增版本统计摘要（总版本数/有效版本/失效版本）。
💾 导入/导出弹出备份窗口，可导出全部数据（含操作日志）为 JSON 文件，或从备份文件恢复数据，支持日志追加/替换模式。
📋 操作日志弹出日志窗口，记录所有关键操作（新增/编辑/删除/收藏/草稿/批量/填入/复制/清空/导入导出等），支持按类型筛选、按时间段折叠（最近7天、8-30天前、30天前）。自动记录操作时的目标网站域名。
❓ 功能说明打开本页面。鼠标悬停可预览摘要。支持截图自动轮播与手动切换。
🔄 版本更新查看版本更新日志。鼠标悬停可预览最新更新内容。
底部信息区位于左侧导航栏最底部，显示插件名称、作者、当前版本（动态读取 manifest.json，无需手动维护）、更新日期，排版分层清晰。NEW

📋 提示词列表（右侧内容区）

顶部工具栏分为四行：

  • 第一行：搜索框 + 排序下拉框（支持7种排序方式，见下文）
  • 第二行：分类筛选 + 标签筛选
  • 第三行：全选、批量删除、批量移动分类（含下拉框和按钮）
  • 第四行：左侧新增提示词、清空输入框；右侧显示总数、置顶数量、收藏数量

排序下拉框选项说明：

选项排序规则是否显示分类标题置顶是否优先
默认排序分类内最新修改在前（按 updateDate 降序）是是（置顶词条排在该分类最前）
分类内：最早修改在前分类内按 updateDate 升序是是
分类内：使用次数最多分类内按 useCount 降序是是
全局：使用次数最多全部词条按 useCount 降序（跨分类）否否
全局：置顶优先全部词条中置顶词条在前，内部按使用次数降序否仅保证置顶整体在前
全局：收藏优先全部词条中收藏词条在前，内部按使用次数降序否否
全局：版本号降序全部词条按语义版本号降序（如 2.0 → 1.5）否否

词条卡片布局：每个词条分为四行。第一行左侧为复选框、置顶按钮、标题及版本号（标题超长时自动截断，版本号与使用次数始终可见），右侧为状态标签（"已置顶"或"已收藏"）；第二行为操作按钮组；第三行为分类及标签信息；第四行为创建和修改时间。NEW

每个提示词卡片上的按钮

卡片右上角会显示"已置顶"或"已收藏"文字标签（置顶优先），帮助快速识别状态。

图标/按钮功能说明
☑️ 复选框勾选后可用于批量删除或批量移动分类（鼠标悬停有提示）。
📌 置顶按钮点击后词条置顶：置顶词条会在其所属分类中排在该分类的最前面（不受时间排序影响），并一定会出现在浮动面板中（但面板内仍按使用次数排序，不因置顶而优先）。按钮图标会切换：未置顶时显示 📌，已置顶时显示 📍。鼠标悬停提示当前状态。卡片右上角显示"已置顶"标签。
📥 填入将提示词内容自动写入当前 AI 网页的输入框，增加使用次数，并记入日志。
📋 复制复制提示词内容到剪贴板，操作记入日志。
☆ 收藏 / ⭐ 已收藏将词条加入"我的收藏"（再次点击取消收藏），操作记入日志。在主列表中，已收藏的词条（非置顶）会显示浅蓝色背景和左侧蓝色边框，卡片右上角显示"已收藏"标签。
📄 转草稿将正式词条移至草稿箱，操作记入日志。
✏️ 编辑打开编辑窗口，可修改标题、内容、分类、标签、备注、版本号等，保存时记录版本变化。
✖ 删除永久删除当前版本（会弹出确认框），操作记入日志。

⭐ 我的收藏 & 📝 草稿箱

这两个页面的卡片按钮与提示词列表类似，但有以下区别：

  • 我的收藏：将"收藏"按钮变为"✖ 取消收藏"。注意："删除"按钮在此页面也会彻底删除词条（从所有列表中移除），并弹出确认框。收藏词条在主列表中会以浅蓝色背景+左侧蓝色边框高亮显示，并显示"已收藏"标签。
  • 草稿箱：将"转草稿"按钮变为"📌 转为正式"。转为正式时，会自动禁用所有同名的旧正式词条（保留在版本历史中），并创建新的正式版本。

📁 分类管理 & 🏷️ 标签管理

用于全局维护分类和标签列表。分类管理新增颜色选择器，可为每个分类指定一种背景色（15种预设颜色）。删除分类或标签不会影响已有提示词（只是不再关联）。新增/删除操作会记入操作日志。

✨ 详细功能特性

🔹 核心功能

  • 提示词管理：新增、编辑、删除、本地永久存储
  • 分类/标签管理：支持自定义分类、多标签点选（编辑页直接切换选中状态）
  • 三大状态：正式列表、收藏夹、草稿箱自由切换，草稿与正式词条可互转
  • 批量操作：批量勾选、批量删除、批量迁移分类 — 复选框悬停有提示，操作自动记录日志
  • 搜索排序：标题/内容/标签关键词搜索，支持7种排序方式（分类内/全局，按时间、使用次数、置顶、收藏、版本号）
  • 置顶：点击 📌/📍 按钮可将词条置顶。置顶词条会在其所属分类中排在该分类的最前面（不受时间排序影响），卡片带有浅黄色背景和左侧橙色边框，卡片右上角显示"已置顶"文字，同时会出现在浮动快捷面板中（带"📍 置顶"标签）。
  • 收藏视觉区分：在主列表中，已收藏的词条（非置顶）会显示浅蓝色背景和左侧蓝色边框，卡片右上角显示"已收藏"文字，方便快速识别。
  • 分类背景色：在分类管理页面可为每个分类设置背景色（15种预设颜色）。主列表中，非置顶且非收藏的词条将自动应用其分类的背景色，并带有同色系加深的左边框，帮助快速区分不同分类的词条。
  • 版本历史：独立页面以时间轴展示每个提示词的所有迭代版本，支持一键恢复旧版为当前生效版本。恢复时，该标题下的所有旧版本会被自动取消收藏和置顶，恢复的版本本身也不带收藏/置顶状态。删除当前生效版本时，若该词条系列下还存在其他历史版本，系统会自动将最新历史版本恢复为生效状态，防止误删导致数据完全丢失。页面顶部新增版本统计摘要。
  • 使用次数统计：每个提示词的使用次数自动累计，高频词条（≥5次）自动出现在浮动面板中，卡片上会显示"🔥使用次数"标签。
  • 操作日志系统：自动记录所有关键操作，支持按类型筛选、按时间段折叠（最近7天、8-30天前、30天前）、清空日志。填入/清空等外部操作自动记录目标网站域名。
  • 左侧导航分组折叠：支持折叠/展开分组，状态自动记忆
  • 分类/标签筛选：顶部工具栏增加分类筛选和标签筛选下拉框
  • 按钮悬停预览：鼠标悬停在"功能说明"、"版本更新"按钮上可预览摘要
  • 浮动面板智能推荐（增强版）：在 AI 网页内按 Alt+1 唤出快捷面板，展示 收藏 + 置顶 + 高频词条（使用次数≥5）。面板新增功能：
    - 排序切换：可在"按使用次数排序"和"置顶优先排序"之间切换。
    - 搜索过滤：实时根据标题或内容过滤词条。
    - 查看更多：初始显示8条，点击按钮可展开至20条。
    - 自动刷新提示：当管理器中数据变化时，面板自动刷新并显示"已更新"提示。
    - 卡片增加微边框和阴影，视觉更清晰。
    排序规则：使用次数优先时按使用次数降序→最近使用降序；置顶优先时所有置顶词条在前，内部按使用次数降序。

🔹 快捷操作

  • 一键填入：将提示词自动写入 AI 网页输入框（深度适配 Lexical 富文本编辑器）
  • 一键复制：快速复制提示词内容到剪贴板
  • 清空输入框（浮动面板内）：面板头部增加「清空」按钮，一键清空当前 AI 输入框
  • 全屏模式：点击左侧导航栏顶部的按钮，可在独立标签页中打开管理界面，获得更大操作空间。
  • 唤起管理器：按 Ctrl+Shift+L 或点击浏览器图标打开完整管理界面
  • 版本历史按钮：点击左侧「📜 版本历史」按钮，查看所有提示词的版本演进

🔹 编辑增强

  • 标签点选：编辑页以可视化标签按钮展示所有标签，点击即选中/取消，无需手动输入
  • 编辑页新增标签：可实时创建新标签并自动同步到标签管理页
  • 实时字数统计：内容框字符数实时显示
  • 版本号管理：正式词条默认从 0.1 开始，每次编辑保存后版本号自动递增（0.1→0.2→0.3），草稿编辑不递增；旧版本自动保留在版本历史中，支持恢复
  • 同名自动归档：新建与已有词条同名的正式词条时，旧版本自动禁用，仅最新版本生效；编辑正式词条时也会自动创建新版本并保留旧版；草稿转为正式时同样触发此机制。
  • 备注信息：为每条提示词添加备注，鼠标悬浮停留 400ms 后自动预览（快速划过不闪烁）

🔹 数据备份

  • 导出增强：一键导出全部数据（正式词条、草稿、收藏、分类、标签、使用次数统计、操作日志）为 JSON 文件，文件名包含精确到秒的时间戳
  • 导入增强：支持选择本地 JSON 备份文件直接恢复数据，导入时可选择"追加日志"或"替换日志"模式
  • 双存储机制：localStorage + chrome.storage 实时双向同步，确保浮动面板与管理器数据一致

🔹 交互体验

  • 懒加载缓存：子页面切换时保留勾选状态和滚动位置，数据未变更不重复渲染
  • 脏标记刷新：数据增删改后仅标记受影响页面，切回时自动增量更新
  • 实时数据同步：从浮动面板填入后，管理器的使用次数自动刷新，无需手动操作
  • 复选框 Tooltip、置顶按钮 Tooltip（显示当前状态）
  • 左侧导航分组折叠：支持折叠/展开分组，状态自动记忆
  • 分类/标签筛选：顶部工具栏增加分类筛选和标签筛选下拉框
  • 按钮悬停预览：鼠标悬停在"功能说明"、"版本更新"按钮上可预览摘要
  • 弹窗高度动态适配：Popup 模式下自动检测实际视口高度并修正 body/html 高度，彻底消除无法滚动的幽灵滚动条。 NEW
  • 版本号动态同步：底部信息区版本号自动读取 manifest.json，避免手动更新遗漏。 NEW
  • 词条卡片布局优化：操作按钮组独立成行，标题超长自动截断且保留版本号/使用次数始终可见，分类与标签合并为同一行展示，视觉层次更清晰。 NEW
